Conhecimentos

HTML

HTML (Linguagem de Marcação de HiperTexto) é um bloco de construção básico que define o significado e a estrutura do conteúdo da web.

CSS

O CSS (Cascading Style Sheets) é uma linguagem de folhas de estilos utilizada para estabelecer a formatação a qual os documentos escritos na linguagem de marcação (HTML ou XML) devem ser apresentados. Normalmente, enquanto o HTML é usado para estruturar os conteúdos, o CSS é utilizado para formatá-los. Dessa forma, há uma enorme integração entre o HTML e o CSS.

Javascript

JavaScript é uma linguagem de programação que permite a você criar um conteúdo que se atualiza dinamicamente, controlar múltimídias, imagens animadas, entre outras aplicações, o que confere dinamismo ao seu projeto.

Node.JS

Node.JS é um ambiente que permite rodar Javascript no back-end, ampliando as potencialidades dessa linguagem altamente uitlizada.

PHP

PHP (PHP Hypertext Preprocessor) é uma linguagem de script open source muito utilizada, especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.

Bancos de Dados

Serviços de banco de dados são ofertas que armazenam dados e permitem o acesso através de aplicativos externos. Muitas aplicações em todo o mundo utilizam serviços de bancos de dados confiáveis para seus requisitos de gerenciamento de dados. Entre bancos de dados relacionais e não-relacionais, destacam-se o MySQL, PostgreSQL e MongoDB.
